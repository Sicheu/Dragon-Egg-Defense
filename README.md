# 🐉 드래곤 알 지키기 (Dragon Egg Defense)

본 프로젝트는 스타크래프트 2 유즈맵 “랜덤 타워 디펜스”에서 영감을 받아 개발된 Unity 기반 타워 디펜스 게임입니다.  
플레이어는 드래곤의 숲을 침략하는 인간들의 공격을 막아내며, 전략적인 유닛 배치와 스킬 활용을 통해 드래곤 둥지를 방어하게 됩니다.

## 📅 개발 기간
2024.08.01 ~ 2024.09.09 (약 1개월)

## 💻 기술 스택
- Unity 2022.3.30f1
- Rider 2024.1.2
- C#
